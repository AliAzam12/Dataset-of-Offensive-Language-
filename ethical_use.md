# Ethical Use Guidelines

This resource contains offensive and potentially harmful language. It should be handled carefully.

## Allowed uses

The dataset may be used for:

- academic research;
- educational work;
- benchmark evaluation;
- moderation-support experiments;
- low-resource and multilingual NLP research;
- analysis of model robustness and fairness.

## Disallowed or discouraged uses

The dataset should not be used to:

- target, harass, or profile individuals or communities;
- generate abusive or hateful content;
- create tools that amplify harmful language;
- make unsupported claims about any group or language community;
- deploy fully automated moderation without human review;
- train models for surveillance or discriminatory profiling.

## Researcher responsibilities

Users should:

- cite the dataset and associated paper;
- verify source dataset licences;
- avoid publishing unnecessary offensive examples;
- anonymise or mask personal identifiers where required;
- report language-wise and class-wise results;
- discuss limitations and possible bias;
- use human review for real-world moderation settings.

## Exposure control

When presenting examples in papers, slides, or demos:

- use only the minimum number of examples needed;
- mask slurs where possible;
- warn readers before showing harmful content;
- avoid repeating offensive content in public-facing material.
