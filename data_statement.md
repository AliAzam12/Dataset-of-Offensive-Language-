# Data Statement

## Dataset name

Multilingual Offensive Language Resource and Benchmark for Low-Resource South Asian Social Media

## Languages

|Language|Script/writing condition|
|-|-|
|Urdu|Arabic script|
|Roman Urdu|Latin/Romanised script, often code-mixed with English|
|Pashto|Arabic-derived Pashto script|
|English|Latin script|

## Text type

The dataset contains social media text intended for offensive-language detection research.

## Label type

Binary classification:

* `offensive`
* `non-offensive`

## Linguistic characteristics

The dataset includes several forms of noisy user-generated text:

* transliteration variation;
* informal spelling;
* code-mixing;
* repeated characters;
* hashtags;
* emojis or informal symbols;
* dialectal and low-resource language variation;
* platform-specific writing conventions.

## Known limitations

* Some source datasets may use different annotation schemes.
* Label harmonisation can reduce fine-grained distinctions between hate speech, abuse, toxicity, profanity, and insult.
* Code-mixed and Romanised text can contain spelling variants that are difficult to normalise.
* Pashto and Urdu subsets may not cover all dialects or regional varieties.
* The dataset should not be treated as a complete representation of any language community.
* Original source licences and redistribution rights must be checked before reuse beyond research settings.
