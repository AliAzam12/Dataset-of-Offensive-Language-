Multilingual Offensive Language Resource and Benchmark

This folder contains the dataset files for Urdu, Roman Urdu, Pashto, and English offensive-language detection.

Task:
Binary offensive-language classification

Labels:
offensive
non-offensive

Exact corpus size:
Total samples: 144,265
Roman Urdu: 73,000
Pashto: 34,400
English: 24,783
Urdu: 12,082

Exact label totals:
Offensive: 85,161
Non-offensive: 59,104

Files:
full data.csv
train data.csv
validation data.csv
test data.csv
dataset_summary_counts.csv

Columns:
id, text, clean_text, language, script, source_platform, source_dataset, original_label, final_label, label_id, split, topic, is_code_mixed, has_emoji, has_hashtag, contains_url, text_length_chars, token_count, created_year

Use:
Research, education, benchmark evaluation, and moderation-support experiments only.

Note:
The dataset contains offensive and potentially harmful language.
